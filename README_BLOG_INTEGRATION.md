# 🎨 Blog Integration System - START HERE

## Welcome! 👋

This is your complete blog integration system for adding beautiful blog sections to **any page** on your main website or minisites.

---

## 🚀 Quick Start (30 Seconds)

### Step 1: Import the Component
```tsx
import { EasyBlogSection } from '@/lib/blog-integration';
```

### Step 2: Add to Your Page
```tsx
<EasyBlogSection 
  preset="miniSiteStandard" 
  loadFromAPI={true}
  title="Latest Blog Posts"
/>
```

### Step 3: Done! ✅
Your blog section is now live on your page!

| Preset | Best For | Code |

| **miniSiteStandard** | Mini-sites, general use | `preset="miniSiteStandard"` |

| **compactGrid** | Sidebars, footers | `preset="compactGrid"` |
