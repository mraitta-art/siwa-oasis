How It Works

  1. User Sees Thumbnail

    High-quality YouTube thumbnail with gold play button overlay

    Thumbnail + Play Button + Title

  2. User Clicks Play

    YouTube IFrame loads and video starts playing (muted by default)

    Video plays → "PLAYING" badge appears → Mute button shows

  3. User Hovers Video

    Custom controls fade in (play/pause + mute/unmute)

    Controls appear on hover → User can pause or toggle sound

  4. User Clicks to Pause

    Video stops, thumbnail with play button returns

    Video pauses → Thumbnail shows → Ready to play again

Integration (3 Steps)

  1. Add Import

    In AdvancedHeroCarousel.tsx, add:

    import YouTubeCarouselPlayer from './YouTubeCarouselPlayer';

  2. Replace YouTubeBackground Function

    Use YouTubeCarouselPlayer instead of YouTubeFacade:

    <YouTubeCarouselPlayer
    videoId={videoId}
    isActive={isActive}
    showControls={true}
    autoplay={false}
    />

  3. Test & Enjoy!

    Run npm run dev and test your custom YouTube player
