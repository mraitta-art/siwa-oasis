❌ Before (Direct IFrame)

<!-- Loads immediately, blocks page --> <iframe src="https://www.youtube.com/embed/VIDEO_ID" width="560" height="315" ></iframe> // Problems: // ✗ Loads on page render (slow) // ✗ 15+ HTTP requests per video // ✗ ~1.5 MB data per embed // ✗ Sets tracking cookies // ✗ Blocks page load

✅ After (Facade Pattern)

<!-- Lightweight thumbnail, loads IFrame on click --> <YouTubeFacade videoId="VIDEO_ID" title="Video Title" thumbnailQuality="maxresdefault" /> // Benefits: // ✓ Loads thumbnail only (~50KB) // ✓ 1 HTTP request initially // ✓ No cookies until interaction // ✓ Non-blocking, async // ✓ Privacy mode enabled

📝 Implementation Code

Vanilla JavaScript Version:

function createYouTubeFacade(container, videoId) { // Create thumbnail wrapper container.innerHTML = ` <div class="youtube-facade-wrapper"> <div class="youtube-facade-thumbnail" style="background-image: url('https://img.youtube.com/vi/${videoId}/maxresdefault.jpg')"> <div class="youtube-facade-overlay"></div> <div class="youtube-facade-play-btn"> <svg viewBox="0 0 24 24"><path d="M8 5v14l11-7z"/></svg> </div> </div> </div> `; // Load IFrame on click container.querySelector('.youtube-facade-wrapper') .addEventListener('click', function() { const origin = window.location.origin; const embedUrl = `https://www.youtube-nocookie.com/embed/${videoId}?rel=0&enablejsapi=1&playsinline=1&origin=${origin}`; this.innerHTML = `<iframe class="youtube-facade-iframe" src="${embedUrl}" allowfullscreen></iframe>`; }); } // Usage: const container = document.getElementById('youtubeFacade'); createYouTubeFacade(container, 'dQw4w9WgXcQ');
