## 💡 Key Concepts at a Glance

### Inheritance
```
Main Hotel (Parent)
  └─ Downtown Hotel (Child)
     Inherits: Basic Data, Amenities, Pricing
     Overrides: Hours (24/7), Phone
     Custom: Location Details
```

### Search
```
Customer: Find all hotels where price < $100
System: Query component_data where 
        component_id = price 
        AND value < 100
Result: [Hotel A, Hotel B, Hotel C]
```

### Comparison
```
Customer: Compare Hotel A vs B vs C
System: Get comparable components from:
        - Pricing section
        - Amenities section  
        - Ratings section
Result: Side-by-side comparison table
```
